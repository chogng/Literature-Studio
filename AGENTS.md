# AGENTS

## Naming Convention
- Use `camelCase` for variables and functions (for example: `articleList`).
- Use `PascalCase` for types, interfaces, classes, and React components.
- Use `UPPER_SNAKE_CASE` for compile-time constants.
- Use `kebab-case` for filenames in `src/` (for example: `article-list.ts`).
- Keep naming consistent across the whole codebase; avoid mixing `snake_case` and `camelCase` in JavaScript/TypeScript code.
